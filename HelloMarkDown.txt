# git 명령어 요약

- Clone : 원격 저장소 복사
- add : 스테이지 영역에 작업파일 추가
- Commit : 스테이지에 올라온 파일을 가지고 내 컴퓨터에
	  세이브, 스테이지 영역의 파일들을 가지고 커밋(==세이브)를 만들 수 있다.
- push : 커밋들을 원격 저장소에 커밋을 업로드 한다.
- CheckOut 을 이용하면 아주 쉽게 마지막 커밋으로 되돌아갈수 있다.
소스트리의 코드뭉치 버리기 기능을 사용하면 변경사항을 되돌릴수 있다.
CheckOut을 이용해서 저장하지 않은 병경 내용 취소하기.
